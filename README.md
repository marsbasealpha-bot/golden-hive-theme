# Golden Hive – Shopify Theme for Beehive-Shop.com 🍯

**Clean • Modern • Honey-focused**

Premium theme for raw honey, beeswax, skincare & gifts. Warm golden colors, elegant typography, subtle honeycomb textures, built-in mega menu, and lightning-fast performance.

## Features
- Honey Gold (#E8B923) + Cream (#FAF4E8) palette
- Playfair Display headings + Inter body
- Built-in mega menu (desktop) + smooth mobile drawer
- Sticky header + announcement bar
- 100% theme-editor friendly

## Quick Install
1. Download this repo as ZIP
2. Shopify Admin → Online Store → Themes → **Add theme** → Upload ZIP  
   OR use "Connect from GitHub" (new Shopify feature)
